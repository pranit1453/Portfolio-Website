## Introduction to Java Fundamentals
Java is a high-level, object-oriented programming language widely used for building enterprise applications, web services, mobile apps, and large-scale systems. It follows the principle of "Write Once, Run Anywhere" (WORA) due to the Java Virtual Machine (JVM).

### Core Features of Java
- **Platform Independent:** Java code runs on JVM, making it portable across operating systems.
- **Strongly Typed:** Variables must be declared with specific data types.
- **Automatic Memory Management:** Garbage Collector handles unused object cleanup.
- **Robust & Secure:** Exception handling and bytecode verification improve security.

## Object-Oriented Programming (OOP) Concepts
Java is built around Object-Oriented Programming principles that promote modular, reusable, and maintainable code.

### 1. Encapsulation
Encapsulation means wrapping data (variables) and methods into a single unit (class). It is achieved using private variables and public getter/setter methods.

### 2. Abstraction
Abstraction hides internal implementation details and shows only essential functionality. It is achieved using abstract classes and interfaces.

### 3. Inheritance
Inheritance allows one class to acquire properties and behaviors of another class using the `extends` keyword. It promotes code reusability.

### 4. Polymorphism
Polymorphism allows methods to perform different behaviors based on the object context. It is achieved via method overloading (compile-time) and method overriding (runtime).
